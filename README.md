# お客様相談管理システム

## 作成した目的

お客様からのご意見をいただく窓口とし、この情報をもとにクレーム対応を行ったり、
今後のサービス向上に役立てるため。

![](imgs/2023-10-05-20-59-42.png)

## アプリケーション URL

http://localhost/

## 機能一覧

お客様からのご意見を、お客様情報と共にデータベースに登録する。
登録されたお客様情報を検索し、該当データを出力する。

## 使用技術（実行環境）

Laravel Framework 8.83.27
